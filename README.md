# How to setup

clone this repository into the home directory. (`~/dotfiles`)

## setup using a symbolic link.

e.g.) `ln -s ~/dotfiles/.vimrc .`

| Source           | Directory      |
|------------------|----------------|
| .vimrc           |  `~`           |
| .vim/vim-include |  `~/.vim`      |
| .hammerspoon     | ~/.hammerspoon |
